# گزارش آسیب‌پذیری‌های امنیتی

اگر یک آسیب‌پذیری امنیتی در این پروژه پیدا کرده‌اید، لطفاً آن را از طریق روش‌های زیر گزارش دهید.

## گزارش دادن

**لطفاً از طریق Issues عمومی گزارش ندهید.**

برای گزارش آسیب‌پذیری‌های امنیتی، لطفاً ایمیل بفرستید به: [dev@khashayar.one]

## چه انتظاری داشته باشید؟

- ظرف 48 ساعت پاسخ دریافت خواهید کرد
- ما مشکل را تأیید و اولویت‌بندی می‌کنیم
- پس از رفع مشکل، شما را مطلع خواهیم کرد

## بهترین روش‌های امنیتی برای کاربران

- هرگز کلیدهای API خود را در فایل‌های کد قرار ندهید
- همیشه از Secrets در GitHub Actions استفاده کنید
- فایل `.env` را در `.gitignore` قرار دهید

با تشکر از کمک شما برای حفظ امنیت این پروژه!
